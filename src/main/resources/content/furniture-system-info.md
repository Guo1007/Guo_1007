## 六、订单系统

### 订单状态说明

| 状态码 | 含义 |
|--------|------|
| 1 | 待付款 |
| 2 | 待发货（已付款） |
| 3 | 已发货 |
| 4 | 已取消 |
| 5 | 已完成 |

### 订单流程
1. 用户下单 → 状态为"待付款"（1）
2. 用户付款 → 状态变为"待发货"（2）
3. 商家发货 → 状态变为"已发货"（3）
4. 用户确认收货 → 状态变为"已完成"（5）

**注意**：用户可以在付款前取消订单（状态变为4），商家也可以在未发货前取消订单。

### 订单信息包含
- 订单ID、用户ID、总金额
- 收货人姓名、联系电话、收货地址
- 备注信息
- 创建时间、付款时间、发货时间、收货时间
- 订单商品明细（商品名称、价格、数量、规格快照）

---

## 七、评价系统

- 用户可以对已购买的商品进行评价
- 评价内容包括：评分（1-5星，默认5星）和文字评价
- 评价记录关联用户ID、订单ID和家具ID

---

## 八、收藏系统

- 用户可以收藏感兴趣的家具商品
- 每个用户对同一商品只能收藏一次（唯一约束）
- 收藏记录包含用户ID、家具ID和收藏时间

---

## 九、通知系统

### 通知类型
| 类型 | 含义 |
|------|------|
| system | 系统通知 |
| order | 订单通知 |
| promotion | 促销通知 |

### 通知规则
- 通知可以发送给全体用户（user_id为NULL）或指定用户
- 系统记录用户的已读状态，支持未读消息提醒

---

## 十、用户系统

### 用户信息
- 手机号、邮箱、用户名、头像
- 收货地址（可设置默认地址）
- 收货人姓名、收货人电话
- 注册时间

### 用户角色
- **普通用户**（is_admin = 0）：浏览商品、下单、收藏、评价等
- **管理员**（is_admin = 1）：商品管理、订单管理、用户管理、通知管理等

---

## 十一、常见问题回答指引

### 关于产品
- 所有家具品牌均为"顾家"
- 价格区间：¥380（餐椅）至 ¥5,680（真皮沙发）
- 实木餐桌支持黑色和白色两种颜色规格，价格不同

### 关于购买
- 用户需要先注册登录才能下单
- 支持多收货地址管理
- 订单支持备注信息

### 关于售后
- 用户可以在发货前取消订单
- 收到商品后可以进行评价
- 有问题可通过通知系统联系客服

### 关于配送
- 订单生成后商家安排发货
- 发货后用户可在订单详情中查看物流状态
- 用户确认收货后订单完成
